## 1.1 О проекте Sedona

Sedona - это небольшой туристический городок в штате Аризона, США. Наша задача - создать привлекательный одностраничный сайт, который покажет преимущества этого места перед знаменитым Гранд-Каньоном.

### Основные разделы сайта:
1. **Шапка** с навигацией и логотипом
2. **Приветственный блок (Hero)** с красивым изображением
3. **Преимущества** - почему стоит посетить Sedona
4. **Поиск гостиниц** - призыв к действию
5. **Подписка на рассылку** - форма подписки
6. **Подвал** с контактами и социальными сетями

## 1.4 Создание базового HTML файла

Создайте файл `index.html` со следующей структурой:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Sedona - Городок в Аризоне</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <!-- Здесь будет содержимое страницы -->
    <h1>Sedona</h1>
</body>
</html>
```

### Разбор кода:

1. **`<!DOCTYPE html>`** - объявление типа документа HTML5
2. **`<html lang="ru">`** - корневой элемент с указанием языка (русский)
3. **`<meta charset="UTF-8">`** - кодировка UTF-8 для поддержки кириллицы
4. **`<meta name="viewport"...>`** - для адаптивности на мобильных устройствах
5. **`<title>`** - заголовок страницы в браузере
6. **`<link rel="stylesheet">`** - подключение файла стилей

## 1.5 Создание базового CSS файла

Создайте файл `style.css`:

```css
/* Общие стили */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: "PT Sans", Arial, sans-serif;
    font-size: 18px;
    line-height: 21px;
    color: #333333;
    background-color: #f2f2f2;
}
```

### Разбор кода:

1. **`* { ... }`** - универсальный селектор, сбрасывает отступы
2. **`box-sizing: border-box`** - упрощает расчет размеров элементов
3. **`font-family`** - основной шрифт для всей страницы
4. **`background-color`** - фоновый цвет страницы

3. Откройте Developer Tools (F12) и проверьте:

   - Нет ли ошибок в консоли

## 1.7 Лучшие практики организации проекта

### Именование файлов:
- Используйте строчные буквы
- Используйте дефисы вместо пробелов
- Используйте понятные имена: `hero-background.jpg`, а не `img1.jpg`

### Структура папок:
```
project/
├── index.html
├── css/
│   └── style.css
├── images/
│   ├── icons/
│   ├── photos/
│   └── backgrounds/
└── fonts/
```

### Комментарии в коде:
```html
<!-- Шапка сайта -->

    ...
</header>
```

```css

body {
    ...
}

/* Стили для шапки */

    ...
}
```
